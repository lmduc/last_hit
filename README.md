# LastHit

Help you to run relevant tests

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'last_hit'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install last_hit

## Usage

- Run current modified tests
`bundle exec rake last_hit:modified_tests`
